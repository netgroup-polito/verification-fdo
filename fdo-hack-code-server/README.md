# FDO-hack-code

To run the attacks involving a malicious Rendezvous Server and a malicious Owner, use the code provided in this folder. Refer to the official FIDO documentation at the following link for instructions on how to run the various components:

[https://github.com/fido-device-onboard/pri-fidoiot](https://github.com/fido-device-onboard/pri-fidoiot)


The code in this folder includes:
* The malicious Rendezvous Server used for **Test 2** in the paper, and
* The malicious Owner used for **Test 3**.
