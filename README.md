# verification-fdo

This repository contains the material accompanying the paper “FDO Protocol: A Possible Solution to Protect the Ownership Voucher Against an Untrusted Supply Chain.”

Below is an overview of the repository structure:

* *proverif/* – Contains the ProVerif source files used for the formal security analysis.

* *FDO_demo_instructions.md* – Provides detailed instructions for running the attacks on the reference FDO implementation.

* *fdo-hack-code-server/* – Includes the code for the malicious Rendezvous Server and malicious Owner components.
